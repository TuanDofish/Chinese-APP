# Bộ biểu đồ thesis-style

Các file trong thư mục này đã được chỉnh theo kiểu báo cáo luận văn:
- Đơn sắc, dễ nhìn, ít rối
- Biểu đồ hoạt động theo luồng dọc
- Biểu đồ tuần tự có tiêu đề và phân vai `actor / boundary / control / database`
- Biểu đồ lớp chia theo từng phân hệ

Khi render bằng PlantUML, giữ nguyên các file `style-sequence.puml`, `style-activity.puml`, `style-class.puml` trong cùng thư mục.
